About

The reason why I chose to research the density of the US population is because I am fascinated by the fact all of our states are so different in so many ways, including how densely compact their State is. We all live in New York City, and while we know that this city is incredibly dense with people, the rest of the state is much more spread out with land and is filled with more rural areas and communities.

1. I retrieved data from US population density Wikipedia page. I imported the csv and json modules to read CSV files (data.csv) and to write JSON files.

  • 2. Then the file paths were defined for the input CSV file (csv_file_path) and the output JSON file (json_file_path).
  • 3. I initialized an empty dictionary named data_dict to hold the data from the CSV file.
  • 4. I then opened the CSV file in read mode.
  • 5. The CSV file is read using the readlines() function to read all lines of the CSV file into a list.
  • 6. I processed each line in the CSV file, except the header line, and each line is placed in a loop. Each line is stripped of leading or trailing whitespace and split into geography and people_mile for server.py
  • 7. I then wrote the dictionary to a JSON file using the json.dump() function to create my server.py and jsonparser.py.
  • 8. I used a python script to parse the data into python dictionaries.
